# lava-cli

The `lava` command-line tool. See the [top-level README](../../README.md) for installation and general usage.

## Subcommands

- `format` (alias `fmt`) — Format Magma source files
- `highlight` (alias `hl`) — Syntax-highlighted HTML output (planned for v0.2)
- `parse` — Pretty-print the syntax tree (planned for v0.2)

## Testing

```bash
cargo test -p lava-cli
```
